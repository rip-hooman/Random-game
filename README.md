# Random Game

A fun console-based puzzle game in Python! Solve riddles, decode ciphers, and guess passwords to progress through multiple levels. Perfect for beginners learning Python or anyone who enjoys a text-based adventure.

# Overview
 ### Random Game is a multi-level text adventure where each level challenges your logic, vocabulary, and decoding skills. The game includes:
 - Password puzzles

 - Riddles and logic challenges

 - Number-to-letter and cipher puzzles

 - Color-coded feedback using colorama for correct (green) and incorrect (red) answers 

# Requirements

- Python 3.x

- colorama library

Install dependencies with pip:

```bash
pip install colorama
```

# How to Run

Clone the repository:

`git clone https://github.com/rip-hooman/Random-game.git`


Navigate into the project folder:

`cd Random-game`


Run the game:

`python main.py`


> Windows users: The console will clear automatically for a smooth experience.

## How to Play

Read the instructions for each level carefully.

Type the correct password or solve the puzzle to advance.

If you enter a wrong answer, feedback will appear in red, and you can try again.

Progress through all levels to complete the game!
